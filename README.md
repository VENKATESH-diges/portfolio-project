# Portfolio Project

This is a Django portfolio web application developed to showcase my projects and allow users to submit reviews.

## Features

- User Signup
- User Login
- home page
- Submit Reviews
- View All Reviews
- Password Reset

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Project Structure

portfolio/
│
├── users
├── reviews
├── account
├── templates
├── static
└── manage.py
